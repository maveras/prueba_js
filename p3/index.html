Pregunta 3

Se pide modelar el siguiente problema: En una sala de clases hay alumnos, cada alumno tiene notas y un promedio final. Se pide: Crear la función constructora alumno que recibe el nombre de éste y un arreglo con las notas La función constructora debe revisar que el nombre sea un string y que el arreglo de notas sean integers Crear el método promedio que devuelva el promedio de notas de todos los alumnos.
