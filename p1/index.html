Pregunta 1

¿Cuál es el problema con el siguiente código? var alumno1 = {nombre: "Matías", edad:5} var alumno2 = alumno1 alumno2.nombre = "Sebastián"

Respuesta

Ambas variable alumno1 y alumno2 apuntan al mismo espacio en la memoria, esto quiere decir que si se modifica una, la otra tambien se vera afectada
